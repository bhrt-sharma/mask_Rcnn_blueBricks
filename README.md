# mask_Rcnn

This repo contains code to Use Mask Rcnn on Kangaroo Dataset ( of 180 images )

For this we use MatterPort Mask R-CNN : https://github.com/matterport/Mask_RCNN

link for annotations and images :
https://github.com/experiencor/kangaroo


Step 1: Clone the Mask R-CNN repository
git clone https://github.com/matterport/Mask_RCNN.git
cd Mask_RCNN
$ python setup.py install

Step 2: Download the pre-trained weights for COCO model from MatterPort.https://github.com/matterport/Mask_RCNN/releases/download/v2.0/mask_rcnn_coco.h5

Step 3: run the python notebook!
